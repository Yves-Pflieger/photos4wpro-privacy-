Collecte des données personnelles

Yves Pflieger, auteur de l'application s'engage à protéger votre vie privée. L'application ne collecte aucune information personnelle sur les utilisateurs ni ne transmet aucune données personnelles. L’application accède uniquement aux fichiers que vous sélectionnez explicitement via le sélecteur Windows. Aucun autre fichier n’est analysé ou transmis.

Confidentialité des avis

Vos commentaires dans le Microsoft Store restent anonymes. Le Store n’affiche jamais votre nom complet. Si un prénom apparaît, il s’agit uniquement d’un pseudo par défaut généré par Microsoft. Votre identité réelle n’est jamais visible pour les autres utilisateurs ni pour le développeur.

Localisation

Cette application n'utilise pas la géolocalisation pour identifier où vous vous trouvez.

Accès à Internet

L'application nécessite un accès à Internet uniquement pour se connecter à Microsoft Store, envoyer un commentaire technique ou acquérir un autre logiciel de l'auteur.

Elle utilise également Internet pour télécharger des cartes depuis le service OpenTopoMap et pour effectuer des recherches de lieux (géocodage) à partir de mots‑clés via le service Nominatim d’OpenStreetMap. Aucune donnée personnelle n’est transmise lors de ces requêtes.

Changement de la politique de confidentialité

L'auteur se réserve le droit de modifier, le cas échéant, ce document. Cependant, si d'éventuelles informations personnelles devaient être utilisées autrement que spécifiées dans l'information juridique fournie au moment de la collecte des données, les utilisateurs seraient informés par le biais d'un message affiché dans l'application elle-même.
